## 🧠 Methodology  

| Step | Description |
|------|--------------|
| 1️⃣ **Data Collection** | Sales, product, and regional demand data provided by Seema Scientific. |
| 2️⃣ **Data Cleaning** | Removed duplicates, handled missing values, and standardized categorical data. |
| 3️⃣ **Feature Engineering** | Extracted KPIs such as Average Order Value, Revenue Growth Rate, etc. |
| 4️⃣ **Exploratory Analysis** | Created Python notebooks for trend visualization and pattern identification. |
| 5️⃣ **Dashboard Creation** | Designed Power BI dashboard integrating dynamic visuals and slicers. |
| 6️⃣ **Deployment** | Website deployed via GitHub Pages; insights hosted in this repo. |
