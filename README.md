## 📖 Overview  

This repository documents the **official website** and **business insights dashboard** created for **Seema Scientific**, a company specializing in laboratory materials and educational equipment for schools and colleges.

## 📊 Business Insights Delivered  

### 📍 Product Sales Overview  
- Identified Lab Glassware to be the top-performing product categories contributing **27%** of total revenue.  
- Seasonal sales trends show a **35% increase** in scientific instruments during academic admission periods.

### 📍 Customer Segmentation  
- Clustered customers into **institutional** vs **individual** buyers using K-Means.  
- Regional segmentation highlighted high demand zones in *Pune*, *Lonavala* and *Mumbai*.
  

### 📍 Forecasting & Prediction  
- Applied ARIMA and Linear Regression for monthly sales prediction.  
- Forecasted **Q4 demand** within ±3% of observed values.

## 🧠 Methodology  

| Step | Description |
|------|--------------|
| 1️⃣ **Data Collection** | Sales, product, and regional demand data provided by Seema Scientific. |
| 2️⃣ **Data Cleaning** | Removed duplicates, handled missing values, and standardized categorical data. |
| 3️⃣ **Feature Engineering** | Extracted KPIs such as Average Order Value, Revenue Growth Rate, etc. |
| 4️⃣ **Exploratory Analysis** | Created Python notebooks for trend visualization and pattern identification. |
| 5️⃣ **Dashboard Creation** | Designed Power BI dashboard integrating dynamic visuals and slicers. |
| 6️⃣ **Deployment** | Website deployed via GitHub Pages; insights hosted in this repo. |
